## Production in the Short Run and the Long Run 

- Ø **Variable input** : can be adjusted over the time period being considered 

- Ø **Fixed input** : cannot be adjusted over the time period being considered 

- Ø **Short run** : a period of time over which one or more inputs is fixed 

- Ø **Long run** : a period of time over which all inputs are variable 

   - Ø Length of long run depends on the production process being considered: 

Auto manufacturer may need years to build a new production facility but software firm may need only a month or two to rent and move into a new space

## Types of Costs 

Ø : **Short run total cost (TC)** 

Ø **Variable cost (VC)** : costs of inputs that vary with the firm’s output level 

Ø **Fixed cost (FC)** : costs of inputs whose use does not vary with the firm’s output level and cannot be varied 

## Types of Costs 

ØVariable cost function: _Variable Cost = VC(Output)_ ØFixed cost function: _Fixed Cost = FC_ ØTotal cost function (includes fixed costs): _Total Cost = FC + VC(Output)_
