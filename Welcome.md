# Welcome to the LLM Wiki

This vault is structured as an **LLM Wiki** — a two-layer knowledge system designed for AI-assisted learning and retrieval.

## How It Works

| Layer | Path | Purpose |
|-------|------|---------|
| **Raw** | `Raw/Sources/` | Original lecture notes, slides, and course material |
| **Wiki** | `Wiki/` | Compiled, concise, linked knowledge notes |
| **Schema** | `Schema/` | Rules, frontmatter specs, and manifests |

## Quick Start

1. **Search first**: query `Wiki/catalog.jsonl` before browsing raw sources
2. **Browse by topic**: start at [[Wiki/index]]
3. **Add material**: place new sources in `Raw/Sources/`, then ingest
