# Descriptive Statistics

## Definition

Descriptive statistics are methods for organizing, summarizing, and presenting data. They reduce large datasets to interpretable summaries without making inferences about a larger population.

## Key Measures

### Central Tendency
- **Mean** (μ or x̄): arithmetic average — sum of values divided by count
- **Median**: middle value in ordered data — robust to outliers
- **Mode**: most frequently occurring value

### Dispersion
- **Variance** (σ²): average squared deviation from the mean
- **Standard Deviation** (σ): square root of variance — same units as data
- **Range**: difference between maximum and minimum values
- **IQR**: interquartile range (Q3 − Q1) — robust spread measure

### Shape
- **Skewness**: asymmetry of the distribution (left-skewed vs right-skewed)
- **Kurtosis**: tail heaviness relative to a normal distribution

## Visual Tools

- Frequency tables and histograms
- Box plots (five-number summary)
- Scatter plots (for bivariate data)
