# Decision Analysis

## Definition

Decision analysis is a systematic, quantitative approach to making choices under uncertainty. It combines probability theory with payoff analysis to identify the action that maximizes expected value.

## Framework

1. **Identify alternatives**: list all possible actions
2. **Identify states of nature**: list all possible outcomes
3. **Estimate probabilities**: assign likelihoods to each state
4. **Compute expected payoffs**: E(payoff) = Σ pᵢ × payoffᵢ
5. **Choose the alternative** with the highest expected payoff

## Key Concepts

- **Prior probability**: initial assessment before collecting data
- **Posterior probability**: revised assessment after observing signals/data
- **Signal**: informative data that updates beliefs (Bayesian updating)
- **Independence**: when a signal is uninformative (P(X|Y) = P(X))
