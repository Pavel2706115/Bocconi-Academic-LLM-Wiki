# Portfolio Theory

## Definition

Portfolio theory (Markowitz, 1952) shows how rational investors construct portfolios to maximize expected return for a given level of risk through diversification.

## Key Principles

- **Diversification**: combining assets reduces portfolio risk if they are not perfectly correlated
- **Portfolio return**: weighted average of individual asset returns
- **Portfolio risk**: depends on individual variances AND covariances between assets
- **Efficient frontier**: set of portfolios offering the highest return for each level of risk

## Risk Decomposition

- **Systematic (market) risk**: cannot be diversified away (β)
- **Unsystematic (idiosyncratic) risk**: can be eliminated through diversification
- Only systematic risk is compensated in equilibrium
