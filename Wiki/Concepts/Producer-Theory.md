# Producer Theory

## Definition

Producer theory studies how firms transform inputs into outputs, minimize costs, and maximize profits. It provides the supply-side foundation for market analysis.

## Production

- **Production function**: Q = f(L, K) — output as a function of labor and capital
- **Marginal product**: additional output from one more unit of input
- **Diminishing marginal returns**: beyond some point, each additional unit of input yields less additional output

## Costs

- **Total cost** = Fixed costs + Variable costs
- **Marginal cost (MC)**: additional cost of producing one more unit
- **Average total cost (ATC)**: TC / Q — U-shaped due to spreading fixed costs then diminishing returns
- **Long-run**: all costs are variable; firms choose optimal plant size

## Profit Maximization Rule

Produce where MC = MR (marginal revenue). In perfect competition, MR = Price.
