# Equity Valuation

## Definition

Equity valuation determines the intrinsic value of a company's stock based on expected future cash flows to shareholders.

## Dividend Discount Model (DDM)

P₀ = Σ DIVₜ / (1+r)^t

### Gordon Growth Model (Constant Growth)

P₀ = DIV₁ / (r − g)

Where g = Plowback Ratio × ROE

## Present Value of Growth Opportunities (PVGO)

P₀ = EPS/r + PVGO

- **Growth stocks**: high PVGO component
- **Income stocks**: stable dividends, low PVGO

## Free Cash Flow Valuation

- Values the entire enterprise using Free Cash Flow (FCF)
- PV = Σ FCFₜ/(1+r)^t + Horizon Value/(1+r)^H
- Horizon value often estimated using constant growth model
